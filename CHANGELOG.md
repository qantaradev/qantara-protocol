## [0.1.0] - 2025-01-XX

### Added
- Initial release of Qantara Protocol
- Smart contract with `register_merchant()` and `settle()` instructions
- Security validations (8 comprehensive checks)
- Jupiter integration for token swaps
- API server with quote and transaction building
- Checkout UI with wallet integration
- Comprehensive test suite
- Documentation

### Security
- Protocol pause mechanism
- Merchant freeze capability
- Payout wallet validation
- Buyback mint validation
- BPS bounds validation
- Jupiter router allowlist
- Slippage protection
